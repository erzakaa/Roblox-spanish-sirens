# Siren V3

Simple Roblox script collection.

## Added script

- `car_panel_spanish_sirens.lua`: creates an in-game car control panel in Spanish with:
  - Siren toggle (random Spanish-style emergency siren sound)
  - Emergency light toggle
  - Speed up / slow down controls
  - Hide/show panel buttons

### Usage

1. Create a **LocalScript** in `StarterPlayerScripts` (or `StarterGui`).
2. Paste the content of `car_panel_spanish_sirens.lua`.
3. Join play mode, sit in a `VehicleSeat`, and use the panel.
